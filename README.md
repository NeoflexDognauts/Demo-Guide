# Default Predict Demo

Данный репозиторий содержит демонстрационный ML-проект для платформы Neoflex Dognauts. Проект включает все необходимые артефакты для воспроизведения сквозного демонстрационного сценария: от обучения модели до её эксплуатации и мониторинга.

## Что демонстрируется 

В рамках сценария последовательно выполняются следующие этапы:

+ Обучение модели в Jupyter Notebook.
+ Логирование экспериментов и регистрация модели в MLflow.
+ Развертывание online-сервиса.
+ Пакетное применение модели.
+ Настройка мониторинга данных.

## Как использовать репозиторий

Для прохождения демонстрационного сценария выполните следующие шаги:

1. Ознакомьтесь с руководством: 
   - *Demo-Guide.md* – версия для работы в браузере. Содержит описание требований к инфраструктуре, полный порядок выполнения шагов, а также описание настройки сервисов. 
   - *Demo-Guide.pdf* – версия того же руководства для офлайн-просмотра или печати.
2. Загрузите необходимые артефакты из репозитория:
   - Архив *DPREDICT_project.zip* с тестовым проектом.
   - Тренировочные (*train.csv*) и тестовые (*test.csv*) наборы данных.
   - Ноутбук *default-predict-demo.ipynb* для обучения модели.
   - Ноутбуки *baseline-pipeline.ipynb* и *primary-pipeline.ipynb* для подготовки данных.
3. Воспроизведите сценарий, последовательно выполняя шаги из Руководства. 

> **Примечание:** Руководство актуально для платформы Neoflex Dognauts версии `2.58.0` и выше.
